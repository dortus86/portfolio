Habilidades

Clique com o mouse nas imagens abaixo para ler

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele estrutura o conteúdo por meio de elementos como títulos, parágrafos, links, imagens e tabelas.

CSS (Cascading Style Sheets) é a linguagem usada para definir o estilo visual de páginas web. Com ela, é possível controlar cores, fontes, espaçamentos, alinhamentos e animações, melhorando a experiência e o design da interface.

FIGMA é uma ferramenta de design colaborativo baseada na web. Usada para criar interfaces, protótipos e sistemas de design, ela permite que equipes trabalhem juntas em tempo real, facilitando o processo de criação visual.

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web. Com ela, podemos criar animações, validar formulários, manipular elementos em tempo real e integrar APIs externas ao site.

REACT é uma biblioteca JavaScript desenvolvida pelo Facebook para construir interfaces de usuário. Utilizando componentes reutilizáveis e estados, ela facilita a criação de aplicações web dinâmicas e responsivas.

Tailwind é um framework utilitário de CSS que permite estilizar elementos diretamente no HTML com classes pré-definidas. Ele acelera o desenvolvimento ao eliminar a necessidade de escrever CSS personalizado do zero.
